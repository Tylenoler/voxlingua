# 产品需求文档 (PRD)

## 项目名称
**VoxLingua** — Vox（声音）+ Lingua（语言）
> 基于大模型的 AI 语音口语对练系统

## 1. 产品愿景
打造一个**英语口语为核心、纽约口音可定制、带智能纠正**的 AI 语音对练系统。

你在电脑上跑所有语音模型（STT/TTS/声纹克隆/发音评分），用手机做便携麦克风终端，在日常聊天对话中跟 AI 进行真实、自然的对话，AI 会利用云端大模型的对话能力让交流更真实，同时实时纠正你的发音。

## 3. 核心功能

### 【P0 — 核心】日常口语对练（Daily Chat）

**对话流程：**
```
[手机] 用户用英语说话
   → [WiFi] 音频发送到电脑
      → [电脑] Whisper STT → 文字
         → [电脑/云端] LLM 生成日常对话回复
            → [电脑] CosyVoice 用【纽约口音】合成语音
   → [WiFi] 音频返回手机
[手机] 播放 → 同时显示文字 + 纠正提示 → 用户继续 → 循环
```

**功能点：**
- **日常闲聊场景**：天气、爱好、今天过得怎样、美食、旅行等
- **沉浸式对话**：AI 模拟一个纽约本地人跟你聊天
- **对话中实时纠正**：用户说错的地方，AI 在回复中自然带出正确表达
- **对话历史**：完整保存，可回顾复盘

### 【P1 — 重点】发音纠正系统（Pronunciation Correction 🎯）

**这是本项目最核心的模块。**

```
用户录音
   ↓
Whisper STT → 识别文本
   ↓
Forced Alignment → 音素级时间戳（哪个音在哪个时间点）
   ↓
声学特征对比 → 目标口音 vs 用户发音
   ↓
三级纠正反馈：
   ├── 🟢 轻微错误 → 对话中自然重复正确发音
   ├── 🟡 中等错误 → 手机端实时显示"注意这个发音"
   └── 🔴 严重错误 → 打断对话，播放正确发音示范，用户跟读

   ↓
多维评分 → 发送到手机展示
```

**纠正维度：**
| 维度 | 说明 |
|------|------|
| 🎯 **音素准确度** | 每个音素是否发对（θ→s 这种替换） |
| 📢 **重音位置** | 单词重音是否正确（如 RECord vs reCORD） |
| 🌊 **语调曲线** | 句子升调/降调是否自然 |
| ⚡ **语速** | 是否过快/过慢 |
| 🔗 **连读** | 是否自然连读（如 "gonna"、"wanna"） |

**纠正反馈形式（手机端）：**
```
你：I __ink this is __ good idea.
        ↑            ↑
       [θ→s]       [n→没有]
       
建议：注意 "think" 的 th 音，舌尖要放在牙齿中间
      [🔊 点击收听正确发音]
```

### 【P1 — 重要】纽约口音声纹

- 你提供一段**纽约口音**的参考音频（3-10秒）
- CosyVoice 克隆该声纹作为默认发音教练
- 所有 TTS 输出都使用该纽约口音
- 后续可添加更多口音模板

### 【P2 — 增值】单词练习（Vocabulary Practice）

- **纠正跟读**：AI 读单词 → 你跟读 → 音素级纠错
- **易错词本**：对话中频繁出错的单词自动收录
- **连读训练**：专门训练 "gotta"、"lemme"、"whaddaya" 等口语连读

### 【P3 — 多语言扩展】

- 英语 → 日语 → 中文（后续）
- 每个语言可配置不同的口音模板

## 4. 技术栈

| 层级 | 技术选型 |
|------|---------|
| **桌面壳层** | Tauri v2 (Rust) |
| **桌面 UI** | React + TailwindCSS |
| **AI 引擎后段** | Python + FastAPI |
| **移动端** | Flutter（简约 Material You 风格） |
| **连接** | WebSocket + mDNS |

### 模型选型

| 模块 | 选型 | 位置 |
|------|------|------|
| **语音识别 STT** | Whisper small (multilingual) | **本地** |
| **对话 LLM** | 云端 API（GPT-4o / Claude 等，可配置） | **云端** |
| **回退 LLM** | Qwen2.5-7B (GGUF Q4) | **本地** |
| **语音合成 TTS** | CosyVoice-300M | **本地** |
| **声纹克隆** | CosyVoice (纽约口音模板) | **本地** |
| **发音评分** | 自研音素对齐 + 声学对比 | **本地** |

> **LLM 云端 API 策略**：对话质量比本地 7B 模型自然很多，日常聊天体验更真实。同时保留本地 LLM 作为离线回退选项。

## 5. 显存优化

由于移除本地 LLM 的 5GB 占用（默认使用云端 API），显存压力大幅降低：
- Whisper STT: ~2GB
- CosyVoice TTS: ~2.5GB
- 发音评分: ~1GB
- **总计峰值**: **~5.5GB**（6-8GB 显卡完全无压力）
- 纯 CPU 回退：支持，速度会慢一些

## 6. 里程碑

- [ ] Python 引擎：STT + 云端 LLM + CosyVoice(纽约口音) 核心管线
