# 产品需求文档 (PRD)

## 项目名称
**VoxLingua** — Vox（声音）+ Lingua（语言）
> 基于本地大模型的 AI 语音口语对练系统

## 1. 产品愿景
打造一个**多语言、声纹可定制、本地私密**的 AI 口语陪练系统。用户可以在电脑上运行所有 AI 模型，用手机作为便携交互终端，选择任意真实人物的声纹作为"发音教练"，在真实场景中训练多语言口语。

## 3. 核心功能

### 【P0 — 核心】口语对练（Conversation Practice）

**流程：**
```
[手机] 用户说话录音
    → [WiFi] 发送音频到电脑
        → [电脑] STT 转写文字
            → [电脑] LLM 生成场景化回复
                → [电脑] CosyVoice 用目标声纹合成语音
    → [WiFi] 返回音频到手机
[手机] 播放 AI 回复 → 用户继续对话 → 循环
```

**功能点：**
- 内置丰富的场景模板（餐厅、面试、旅行、商务等）
- 可自定义场景描述和难度级别
- 对话历史持久化保存与回顾
- 支持对话中途切换难度/声纹

### 【P1 — 重要】单词练习（Vocabulary Practice）

- **跟读模式**：AI 用目标声纹读单词 → 用户跟读 → 音素级评分
- **生词本**：对话中不熟悉的单词自动收录，间隔重复复习
- **拼写练习**：听声纹发音 → 拼写单词 → 自动批改

### 【P2 — 增值】发音评分（Pronunciation Scoring）

- **多维评分**：准确度(40%) + 流利度(20%) + 韵律(20%) + 完整度(10%) + 语法(10%)
- **视觉反馈**：波形对比图、错误音素高亮、发音位置热力图
- **历史追踪**：发音进步曲线、薄弱音素统计

### 【P3 — 多语言支持】

- **首发语言**：英语、日语、中文
- **扩展机制**：添加新语言只需配置 STT 模型 + TTS 模型 + 音素表，无需改核心代码
- **Code-switching**：支持中英混说、日英混说等多语言混合场景

## 4. 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **桌面壳层 (Desktop Shell)** | **Tauri v2** (Rust) | 轻量、安全、跨平台（Win/Mac/Linux） |
| **桌面 UI** | **React + TailwindCSS + Radix UI** | 现代、响应式 UI |
| **AI 引擎后端** | **Python Sidecar** (FastAPI) | 通过 Tauri sidecar 启动独立进程 |
| **移动端 (Mobile)** | **Flutter (Dart)** | 跨平台 iOS + Android |
| **连接协议** | **WebSocket + HTTP** | 桌面运行 WS 服务，手机自动发现 |
| **自动发现** | **mDNS (Bonjour/Avahi)** | 零配置局域网发现 |

### AI 模型选型（全部开源，本地运行）

| 模块 | 选型 | 显存占用 | 量化方案 |
|------|------|---------|---------|
| **语音识别 STT** | Whisper small (multilingual) | ~2GB | 可用 ONNX 量化 |
| **大语言模型 LLM** | Qwen2.5-7B-Instruct (GGUF) | ~5GB (Q4) | GGUF Q4_K_M |
| **语音合成 TTS** | CosyVoice-300M | ~2.5GB | FP16 |
| **声纹克隆** | CosyVoice (内置) | 与 TTS 共享 | — |

> **总显存需求**: ~7GB（按需加载，错峰使用，单次仅加载 1-2 个模型）
> **纯 CPU 回退**: 支持，但响应时间会显著增加

## 5. 里程碑规划

### Phase 1 — MVP 原型 (2-3周)
- [ ] Python 侧：STT + LLM + TTS 核心管线跑通（英语）
- [ ] 桌面端：Tauri 壳层 + React UI 雏形 + 启动/停止 AI 引擎
- [ ] 手机端：Flutter 基础 UI + 本地音频录制/播放
- [ ] 手机 ↔ 电脑：局域网 WebSocket 音频流传输打通
- [ ] 跑通完整对话闭环：手机录音→电脑处理→手机播放

### Phase 2 — 功能完善 (2-3周)
- [ ] 发音评分模块
- [ ] 单词练习 + 生词本
- [ ] 声纹模板管理（上传 / 选择 / 预设库）
- [ ] 多语言扩展（日语 + 中文）
- [ ] 对话场景模板库

### Phase 3 — 体验优化 (持续)
- [ ] 发音可视化反馈（波形/频谱对比图）
- [ ] 间隔重复复习系统
- [ ] 对话历史回顾与进度统计
- [ ] 预设名人声纹库（影视/动漫人物等）

## 6. 性能目标
- **端到端延迟**（手机录音 → 收到回复）: < 5 秒
- **显存峰值** ≤ 6GB（适应 6-8GB 显卡）
- **手机端功耗**：低，仅音频编解码 + 网络传输
